Understanding Tonal Values

Tonal values refer to the brightness or darkness of color in an image. They play a crucial role in how artists communicate and how viewers make sense of images. When we remove color from a painting, we're left with the tonal values - the distribution of light and dark areas.

Methods for Organizing Values

Value Studies

A value study is a small sketch used to solve value relationships in a drawing or painting. It should be done quickly and not rendered in detail. The goal is to communicate any image using 4-5 value groups.

Key points for value studies: - Keep it simple and quick - Aim for 4-5 value groups - Use as a map for your final piece

Light Logic

Light logic involves breaking down forms into different light effects:

  1. Light family vs. shadow family (basic two-tone division)
  2. Dark halftones (areas in light turning away from the light source)
  3. Light halftones
  4. Form shadows
  5. Cast shadows

Value Similarity

For complex scenes with multiple local values, use the value similarity method:

  1. Simplify the image by removing color or blurring
  2. Group similar values together
  3. Aim for 4-5 main value groups

Practical Demonstration

Geometric Forms Setup

  1. Start with the darkest value group
  2. Add subsequent lighter value groups
  3. Identify where each group appears in the image
  4. Create soft transitions where appropriate

Portrait Value Study

  1. Begin with a simple line sketch
  2. Establish the darkest value group
  3. Add subsequent value groups, working from dark to light
  4. Keep tones clean and distinct
  5. Adjust and refine relationships between groups

Tips: - Squint to simplify what you're seeing - Use a mechanical eraser and kneadable eraser for adjustments - Work with 2B and 2H pencils for a range of values

Additional Considerations

Working with Diffused Lighting

  • Understand shading and light principles
  • Use light logic to keep things organized
  • Study artists like Bouguereau for atmospheric effects

Texture and Values

  • Do a value study beforehand to establish target values
  • Keep the value study nearby as a reference while adding texture

Smooth Shading Techniques

Use the "island hunting" method: 1. Identify large areas of inconsistent tone 2. Even out these areas 3. Progressively work on smaller inconsistencies 4. Refine until desired smoothness is achieved

* * *

Organizing value structure is crucial for creating effective portraits and realistic drawings. By using value studies, understanding light logic, and practicing value grouping techniques, artists can improve their ability to create convincing and impactful images.
