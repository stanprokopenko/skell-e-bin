OpenAI has had a busy week, and I wanted to share some of their latest developments with you.

GPT-4.1 API Release

On April 14th, OpenAI introduced GPT-4.1 in their API. From my experience, this model significantly improves on coding tasks. I tested it by asking it to solve Advent of Code 2024 Day 3 Part 2 in Rust. Impressively, it one-shotted the problem, generating correct code on the first attempt.

Improved Coding Capabilities

GPT-4.1 outperforms previous versions like GPT-4.0:

  • Higher accuracy: It scores 55% on the SweetBench verified accuracy, surpassing even GPT-4.5.
  • Better tool usage: It follows diff formats reliably and ensures consistent tool usage.
  • Fewer extraneous edits: It makes fewer unnecessary changes, streamlining coding workflows.

Codeex CLI: An Open Source Release

Perhaps more exciting is the release of Codeex CLI, an open-source tool from OpenAI. Unlike other tools that claim to be open but aren't truly accessible, Codeex CLI is genuinely open-source under the Apache License. This means:

  • Commercial use: You're free to use it in commercial projects.
  • Modification and distribution: You can modify, distribute, and even use it privately.
  • Model flexibility: It allows you to use any AI model you prefer.

I've found it functions similarly to other code assistants but with the added benefit of open-source flexibility.

Potential Acquisition of Windsurf

OpenAI is also in talks to purchase Windsurf for $3 billion. Windsurf is a favorite code editor among many developers, and this acquisition could signify OpenAI's commitment to enhancing developer tools. Given that Windsurf users have been testing GPT-4.1 recently, it seems like a strategic move.

Comparing GPT-4.1 and 04 Mini High

I also tested 04 Mini High on coding tasks. While it's a decent model, GPT-4.1 stands out:

  • Efficiency: GPT-4.1 solves problems more directly without over-analyzing.
  • User experience: It provides smoother workflows with fewer iterations.

For example, when asking 04 Mini High to implement a search filter in a React Native app, it worked but required more steps and clarifications. GPT-4.1 handled similar tasks more efficiently.
