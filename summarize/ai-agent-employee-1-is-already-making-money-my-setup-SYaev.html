I've been running an AI agent for the past several weeks, and it's generated over $1,300 in revenue. I wanted to share how I set up this system that automates content creation and brings in passive income.

Building the AI Agent

I used cloud code and a Vercel template to quickly build a simple platform. The AI agent performs data collection, generates parody videos, and promotes content in a subtle way.

The Workflow

Steps Involved

  1. Data Collection: The agent uses the YouTube API to download the latest videos from MoistCr1tikal's channel.
  2. Audio Extraction: It extracts the first 5 seconds of audio using FFmpeg, which serves as an intro.
  3. Transcription and Voiceover: The intro audio is transcribed, and a voiceover script is generated. I use 11 Labs to create a voice that mimics the original.
  4. Video Generation: The agent generates prompts for three 10-second video clips using Replicate API and models like 2.5 Pro and Cling.
  5. Video Assembly: The clips are merged together, ensuring coherence by using the last frame of one clip as the starting point for the next.
  6. Cleanup and Upload: Temporary files are removed, and the video is uploaded to YouTube with autogenerated metadata.

The Results

The generated videos are AI-generated parodies that often turn out amusing. While some are better than others, they consistently engage viewers. The videos include a simple note with a link for those interested in learning more.

Ethical Considerations

I make sure to clearly label the content as AI-generated. Transparency is important, and I avoid misleading viewers. The aim is to entertain, not to deceive.

* * *

This setup has been a great source of passive income with minimal effort after the initial setup. If you're interested in creating similar workflows, I hope this gives you some inspiration. Just remember to be transparent about using AI in your content.
