Understanding Value

Value in art refers to the spectrum of light and dark shades. It's one of the seven elements of art: line, shape, space, form, texture, color, and value. Controlling value is key to creating the illusion of mass, volume, depth, and light in your drawings. Improving your value control will significantly enhance your artwork.

Value Chart Exercise

Create a value chart by drawing several boxes and shading them from lightest to darkest. This chart represents the full range of values your pencil can produce. Focus on consistent, even shading. Use this exercise to develop good habits, like avoiding fingerprints and unintended value changes.

Pressure Scale Exercise

The pressure scale helps you understand how pressure affects value. Start with the darkest value you can make, then gradually reduce pressure to create a continuous gradient to the lightest value. Avoid lifting your pencil or correcting mistakes. Maintain a consistent width to prevent gaps or "tornado" shapes from uneven pressure. Try reversing the exercise, starting from the lightest value.

Value Application Techniques

Hatching

Hatching uses parallel lines to represent value. Closer lines create darker values. Practice laying lines at consistent distances and directions. Concentrate to avoid errors, and if mistakes happen, keep going. To add interest, vary line thickness.

Cross-Hatching

Cross-hatching layers lines in different directions to build values. Plan your value range before starting. Practice on scratch paper to set limits. Be patient and consistent with strokes and directions. Simplify areas by grouping similar values if needed.

Multidirectional Marks

Multidirectional marks create a sketchy finish by varying stroke directions, forming a mosaic of value blocks. You'll need a clear plan and consistent application to prevent distracting variations.

Chiseled Tip Technique

Use a chiseled tip by wearing down your pencil to a flat edge. Apply firm strokes to build value sections, slightly changing direction as needed. Consistent values help maintain the illusion of planes and depth.

Stippling

Stippling uses small dots to create value, common in pen and ink. Add more dots to darken areas. Work from light to dark for more forgiveness. Keep marks consistent to avoid unintended textures. This technique reduces white space to change perceived value. Though time-consuming, it can produce beautiful results.
