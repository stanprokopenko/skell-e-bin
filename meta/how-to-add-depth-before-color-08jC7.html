How to Add Depth Before Color

Now that we've completed the sketch portion of the process, it's time to start laying in some flats, which are the beginnings of getting our values in place. Some people refer to this as applying the local values. Even though there aren't too many different materials here, you'll notice that when we lay them in, the skin color will be a different value than the teeth, and that's going to be really important as we head into the next phase.

Setting Up the Flats

First, I like to organize my layers. I'll grab all the line work I've done and group it using Command/Control + G. If we're going to be good little digital artists, we should name our layers. So, I'll name this group "Line Work".

Next, I make a backup by duplicating the line work group. Then, I'll flatten the duplicate group by hitting Command/Control + E, so now it's just one layer with all the lines. This flattened layer will serve as our primary line work layer.

Now, I'll create a new layer underneath the line work, and start putting in our flats. I call them flats because they're flat colors, with no gradients or shading yet. I'm just going to block out the main shapes with flat values.

Using a Plugin for Faster Flats

I like to use a plugin that my friend Francis Lee and I created. It makes it easier to throw down some shapes quickly. As you can see, we can just use the lasso tool to draw shapes, and it fills them in automatically. If you don't have the plugin, you can do this by creating the shape with the lasso tool, then using the paint bucket tool to fill it in. It just takes a little bit longer, that's why we made the plugin. But for now, this is all we're going to use it for.

Blocking in the Base Shape

I'm going to encapsulate the whole shape of the character. I'll connect the lines to create a closed shape and fill it in. This will be our base shape. I'll keep this in a group and name it "Base". Everything else will be within this group.

Using Clipping Masks

To keep everything within the base shape, we'll use clipping masks. They're super useful. You can hold Alt and click between layers to create a clipping mask, or right-click on the layer and choose Create Clipping Mask.

What clipping masks allow us to do is draw and paint only within the base shape. As we add more layers on top, they stack while using the base shape as a mask. This way, we can keep layering and layering while keeping all the shapes within that one larger shape. We'll be using clipping masks a lot throughout the rest of this tutorial.

Adding Local Values

First, I'll add the eye local values. They're going to be a little bit lighter than the skin color. I'll quickly block in the whites of the eyes and the teeth. These shapes might change later on, but that's fine. Already, even at this stage, it's starting to bring out a little more clarity to what we're working towards.

Now we've got different materials, the teeth are a different material than the eyes, similar but different, and we've got the base skin color. This is a good place to start thinking about our values in terms of lights and darks.

Starting with Shadows

Now that we've got the flats, we're going to explore how to light this thing and make it feel more three-dimensional. I like to start with the shadows, where are my shadows? Where are my darks? That's going to really help solidify the final piece.

I'll create a new layer above the flats and set its layer mode to Multiply. At a simple level, multiply equals darker. There are other layer modes I'll use for lightening things, but multiply is great for darkening.

Laying Down Shadows with Multiply Layers

Using a gray color, I'll use the lasso tool to find those shadow shapes. The multiply layer darkens everything, but the benefit is that it keeps the context of the underlying layers. If we wanted to, say, add an eyeshadow, it doesn't completely wash out the details underneath.

For example, if we just put a color straight over the top with a normal layer, it would block out what's underneath. But with the multiply layer, it keeps everything in context, making it a very versatile layer mode that I use all the time.

I'm thinking about the light source as well, I'll be doing a top-down light source. So the nose will have a shadow here, and it might even cast a shadow onto the teeth. I'll darken the inner mouth, and indicate the eyebrows a bit.

I'll also consider the side planes of the head. If I want less impact, I can use a lighter gray to make a more subtle shadow. By adjusting the brightness of the color on the multiply layer, we can add subtlety to our values.

Adding a Contrasting Background

Another thing I like to do is add a contrasting background. Instead of having a stark white canvas, it's helpful to have a different value in the background to contrast with our subject. This is similar to how traditional artists will stain the canvas.

You could also stain the white background as well. For now, I'll choose a background color that contrasts with the character. We can always change it later.

Refining Shadows and Values

I'll create a soft round brush, nothing fancy, just the default Photoshop brush, and do an overall shadow pass. If our focal point is the head, all of this down here does not need to be super detailed. We can manipulate the eye to focus on the part of interest.

This is called light fall-off. Imagine a spotlight or point light above the character. As light falls off, areas further from the light source become darker.

We can always adjust the opacity of the shadow layer if we've gone too dark. I'll zoom back out to see how we're looking. Turning the shadows layer on and off helps us see the impact.

Merging Layers and Refining Values

Now I've got a good base. This is where I'll do my first merging into one layer, it's easier to refine. I'll duplicate my layers as a backup, then I will merge the working layers together.

I can turn the line work on and off, and see how it's reading. Even at this early stage, it should start to read without the line work, a good sign.

Now, since we've merged these layers, we can make larger changes more easily. I can start to manipulate values further, thinking about where the light source is and how it affects the forms.

Adding Highlights with Screen Layers

Next, I'll introduce another layer mode: Screen. Screen is almost the opposite of multiply. While multiply darkens and keeps context, screen lightens while keeping context.

I'll create a new layer set to Screen mode. Using a lighter gray, I can start adding highlights. For example, I might add highlights to the brow, nose, and cheeks.

I can also use selection tools to help refine these highlights. For instance, using the lasso tool to select an area, I can add a gradient to give the illusion of form.

Iteration and Refinement

At this stage, it's all about iteration. I might go back and forth between normal layers, multiply layers, and screen layers, refining the values and making sure the light source is consistent.

I also like to zoom out and look at the piece from different vantage points. I'm constantly tweaking values, adjusting shapes, and refining details.

I might add occlusion shadows, the darkest parts where light has the hardest time reaching, often in intersections like where the neck meets the jaw.

Adding a Secondary Light Source

We can also add a secondary light source. I'll create another layer set to Screen mode and imagine a light coming from the side or behind, creating a rim light effect. This can help separate the subject from the background.

It's important to add rim lights with intention, not just throwing a thin white line on the edge, but thinking about how the light interacts with the forms.

Adjusting Details

As I refine the painting, I might notice things that need adjusting. For example, the placement of the eyes, the shape of the eyebrows, or the size of the chin.

I can use tools like the smudge tool or transform tools to push and pull shapes as needed. If I want to test different placements, I'll duplicate the layer, make changes, and compare before merging.

I also keep an eye on the values, making sure they're working together. I might squint to see if anything pops out that shouldn't.

Finalizing Values Before Color

Before moving on to color, I want to make sure the values are working well. The more time you spend thinking about these features in the black and white phase, the better off you'll be.

I'll continue tweaking, adjusting forms, and refining details. Once I'm satisfied with the values, we'll be ready to start adding color.

* * *

Now we've got a solid foundation with our values in place. We've taken the initial sketch, added flats, used clipping masks, and built up our shadows and highlights using multiply and screen layers. We've refined the forms and adjusted the details.

In the next phase, we'll start adding color to this black and white painting. Stay tuned as we bring this character to life with color!
