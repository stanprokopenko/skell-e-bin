How to Enhance Drawing Skills with Memory Games

Copy Code

Visual memory is a crucial skill for artists, enabling you to draw from imagination, work faster, boost creativity, and even improve observational drawing. Artists like Karl Kopinski, Kim Jong Gi, Peter Hahn, and Aliza Ivanova owe much of their extraordinary abilities to their well-developed visual memory. While it takes considerable practice to reach their level, these exercises can help accelerate your progress.

Why Visual Memory Matters

Having a robust visual memory helps you in several ways:

  • It allows you to draw from imagination more effectively
  • You can draw faster with a well-trained visual memory
  • It enhances your creativity
  • Surprisingly, it even improves your observational drawing skills

When you have a large mental library of images, you can more easily identify important information in your references. You'll become better at recognizing patterns you've seen before and comparing them with what you're currently observing. This trained visual memory helps you process and store complex information from your references more efficiently while drawing.

Visual Memory Exercises for Beginners

1. Draw from Memory

This exercise helps train your ability to retain visual information:

  1. Look at your reference for 1-2 minutes, studying it carefully
  2. Hide the reference and draw as much as you can from memory
  3. Compare your drawing with the reference to check accuracy
  4. As you improve, try gradually decreasing the time you spend studying the reference

2. Draw from a Different Angle

This game challenges your ability to visualize objects in 3D:

  1. Keep your reference visible
  2. Imagine the subject from a different angle
  3. Draw what you've visualized

3. Combine Memory and Angle Change

This exercise combines the previous two for an added challenge:

  1. Study your reference for a few minutes
  2. Put the reference away and draw it from memory, but from a different angle
  3. For an easier version, start by making a study drawing of the reference, then put it away and draw from memory at a different angle

Advanced Visual Memory Exercises

1. Figure Drawing with Modifications

For more experienced artists, try this challenging exercise:

  1. Use a figure reference as inspiration
  2. Draw the figure from a different angle
  3. Draw from memory
  4. Change the proportions
  5. Use the reference as a starting point to redesign and create your own unique figure

2. Drawing Moving Subjects

This bonus exercise is great for practicing quick visual memory:

  1. Observe a moving subject (like animals at a zoo) for a few minutes
  2. Identify the basic forms and take a mental snapshot of a specific pose
  3. Draw the pose from memory while the subject continues to move
  4. Use ongoing observation to refine proportions and basic forms

Practice Tips

To get the most out of these exercises:

  • Focus on perspective when practicing, especially with 3D forms
  • Construct objects using simple primitives to understand their underlying structure
  • Practice visualizing 3D forms mentally to improve your spatial reasoning
  • Experiment with different focuses in your memory exercises, such as gesture, shapes, colors, or shadows
  • Continue practicing regularly beyond any specific course or assignment to keep improving your visual memory

Remember, developing a strong visual memory takes time and consistent practice. Don't be discouraged if your results aren't perfect at first. The goal is to challenge yourself and gradually improve your ability to visualize and remember complex forms. Keep at it, and you'll see your skills grow over time!
