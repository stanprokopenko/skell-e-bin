Why Visual Memory Is Important

Visual memory is crucial for artists. It helps you draw from imagination, draw faster, be more creative, and even improves your ability to draw from observation. Having a large library of images in your head while you're looking at reference helps you identify the important information. You'll be better at recognizing patterns that you've seen before and comparing those with what you're seeing now.

By having a highly trained visual memory, you'll be better able to process and store the complex information from your reference while you're drawing it.

Visual Memory Games

I'm gonna show you several exercises, challenges, and games that you can play to help improve your visual memory. Artists like Karl Kopinski, Kim Jung Gi, Peter Han, and Eliza Ivanova are able to do what they do because of their extraordinary visual memory. No matter what, it's gonna take a lot of practice to get there. But these visual memory games can help you get there faster.

Game 1: Draw from Memory

Start by looking at a reference for a minute or two. Study it and try to remember as much as you can about it. Then hide your reference and draw as much as you can from memory. If you want, you can gradually decrease the time you spend looking as you get better at this.

After you're done, bring back your reference and check your accuracy. For extra credit, you can try it again to see if you can do it better the second time.

Game 2: Draw from a Different Angle

Look at something and draw it from a different angle. You don't have to hide the reference for this one. Just trying to imagine it from a different angle is enough to improve your ability to visualize.

Game 3: Combine Them

Combine the first two games. Start by looking at your reference for a few minutes, and then draw it from memory and from a different angle. Or you can make it a little easier by starting with a study drawing. Draw it as you see it, then put all that away and draw it from memory and from a different angle. That initial drawing helps you learn the construction of the forms and remember a little bit better than just by looking at it.

Game 4: Advanced Challenge (Level Two)

This one is only for Level Two. If you're drawing from figure reference, draw it from a different angle, from memory, and change the proportions. You're using the reference as inspiration to redesign and create your own thing.

Bonus Game: Drawing from a Moving Subject

This next one's not part of this assignment, but you should try it at some point on your own. Draw from a moving subject. For example, if you go to the zoo, some animals are just sitting around or sleeping, but others are constantly moving.

Start by watching it for a few minutes, figure out the basic forms, and then take a snapshot in your mind of a pose you see as it's moving. Then you draw it. As you're drawing, the animal is still there to help you with the proportions and remind you of the basic forms, but the pose will have to be from memory. Occasionally, if you get lucky, you'll see that pose again.

It's very challenging, but it's a fun exercise that forces you to work with what you've got.
