Level Up Your Shading with Value Scales

Copy Code

Why Make a Value Scale

Creating a value scale teaches you to control the mediums you work with and builds patience. If you can't produce exact values in a simple scale, you can't expect to do it in complex drawings like portraits. It's beneficial to create a value scale for every medium you want to master, whether it's pencil, markers, ink, or charcoal.

For those of you working digitally, I encourage you to try this with a pencil.

Level 1: 11-Step Value Scale

Start by drawing a 1x1 inch rectangle divided into eleven squares. Use a ruler if you like, and draw the lines lightly to avoid dark lines between squares, which can hinder comparison.

Establish your darkest value first using the darkest shade your pencils can produce, this is your value zero. The lightest value is the white of the paper, so you don't need to shade that square.

Fill in the squares between, aiming for a consistent gradation from dark to light. You can start from one end, the middle, or jump around, it's up to you.

Tips for Level 1

  • Use a range of pencils (2H, HB, 2B, 4B, 6B) to achieve the full spectrum of values. Each pencil has a comfortable range.
  • Avoid pressing too hard to achieve dark values, instead, use a softer pencil like a 6B.
  • For lighter values, apply a soft touch with a harder pencil like 2H.
  • Squint at your value scale to see if the gradation is smooth, and adjust as necessary.
  • Clean shading is crucial. Avoid holes and inconsistent patches without using blending tools.

Level 1 Alternative: 5-Step Value Scale

If eleven steps feel overwhelming, start with a five-step value scale. Begin with the darkest and lightest values, then fill in the middle value and the two in-between. This simplifies the gradation and reduces subtlety.

Level 2: Blind Value Scale

For a greater challenge, try creating a blind value scale. You'll shade each square without seeing the others.

How to Create a Blind Value Scale

  1. Cut out one or two-inch squares and number each on the back.
  2. Shade each square to match its assigned value, without comparing it to the others.
  3. Once all squares are done, line them up in order and see how smooth the gradation is.

Adjusting Difficulty

  • Start with fewer squares, like five, to make it easier.
  • For an extra challenge, randomize the order in which you shade the squares.

Optional: Marker Value Scale

If you have markers, try making a value scale with them. Markers like the OLO set with cool gray shades can provide predefined values.

Tips for Markers

  • Applying markers evenly takes practice. Overlapping inconsistently can create streaks.
  • Layering the same marker adds darkness, combining markers can create in-between values.
  • Be aware of drying times, as the true value may not appear until the marker is dry.

* * *

Approach these value scales with patience and take pride in your work. Practicing accurate values pays off in your drawings. Post your value scales for critiques and ensure you take high-quality photos with even lighting, as this will affect the perceived values.
