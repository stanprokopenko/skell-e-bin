How to Create Depth with Values

Value is crucial for creating the illusion of form and three-dimensionality in drawings. In art, value refers to how light or dark something is, not its monetary worth. To understand this concept, look at grayscale images. They demonstrate value differences between elements, like the contrast between skin and lips.

When you're drawing, remember that lines aren't real. They're an abstraction artists use to represent edges of shapes. What we actually see is light affecting forms, perceived as arrangements of colors.

The Components of Color

Color is composed of three elements:

  • Hue: the color family (e.g., yellow, red, blue)
  • Chroma: how pure or gray a color is
  • Value: how light or dark a color is

For artists, value is the most important aspect of color. In fact, many artists categorize value separately from color altogether.

The Role of Value in Drawing

Think of drawing like building construction:

  • Lines are like steel beams (the framework)
  • Values are like walls (creating form and depth)
  • Texture and color are like decorations (adding detail)

As a beginner, it's best to practice drawing without color, focusing on value first. This approach will help you develop a strong foundation in creating form and depth.

The Value Scale

The value scale ranges from 0 (darkest black) to 10 (brightest white). This scale is based on the Munsell Color System, which measures light, not pigment. While the value scale is actually a smooth gradient with infinite values between black and white, artists typically break it down into about 10 steps for practicality.

Keep in mind that not all drawings need to use the full value range. Limited value scales can be used for specific effects, depending on your artistic goals.

Factors Determining Values

Four main factors determine values in art:

  1. Local value
  2. Light on form
  3. Surface material
  4. Atmosphere

As a beginner, you should focus on the first two: local value and light on form.

Local Value

Local value is the inherent lightness or darkness of an object. It's important to note that local value can vary within an object. For example, an apple might have a gradation of values across its surface.

Light on Form

Light on form affects how values are distributed based on the planes of an object. Local value and light on form work together to create the final perceived value. Remember that objects with different local values will have different light and shadow values under the same lighting conditions.

Practicing with Values

Practicing with values is essential for developing your artistic skills. As you continue your art journey, you'll encounter exercises and projects designed to help you apply these value concepts. By mastering value, you'll be able to create more realistic and three-dimensional drawings.
