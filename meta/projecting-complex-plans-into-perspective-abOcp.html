Projecting the Ground Plan

We have a more complex ground plan that we'll turn into elevations. The first thing I do is project the plan. I've placed it so that only one corner touches the surface. That's the only true height line we'll have for this entire object.

This means the height lines for these boxes won't be traceable from there. We'll have to bring them forward to make them touch. The height line for this point of the roof is clear and unforeshortened. The height lines for other points of the roof will have to be brought forward as well.

Placing the Viewer and Vanishing Points

I'll put the viewer back here, placing them right there. This means this is almost a perfect two-point perspective. Our 60-degree cone of vision should include it all, it is very close. It's going to be a wide-angle view.

Now, we want to place our vanishing points. I make sure those vanishing points are going to be in the picture. Here's what I do:

  • I place the straightedge and the triangle and go to the point where the viewer is.
  • From there over to here is the right vanishing point.
  • Aligning with the angles of our viewer, I find the left vanishing point.

Projecting Points to the Viewer

The next job is to take every one of those corners, this one, this one, and so on, and bring them down to the viewer. I make a single dot where the viewer is. If possible, I could put a push pin there, which would make it very exact.

I run lines from each corner to the viewer. This is important. I'm bringing down:

  • Each corner that can be called corners.
  • Points where portions of the building touch each other.

These lines converge at the viewer, helping us project the plan onto the picture plane.

Dropping Vertical Lines from the Picture Plane

Now, we no longer need the viewer. The viewer has established where all these intersections are. The next step is to take the picture plane and pivot it down. This means all these points will now become vertical lines on which we will place height measurements.

I make these vertical lines as thin and unobtrusive as possible. Every one of these lines needs to be precise, they need to touch the dots they're supposed to touch. They may not make sense right now because they're just a bunch of vertical lines, but they will make sense when we're trying to find where a corner may be. We run them down from where the points touch the picture plane.

Establishing the Perspective Grid

Starting with the corner that touches the picture plane, I track it down and decide how low I want it to be on the ground. If we get it too low, it becomes well out of the cone of vision.

We're creating a wide-angle view. In fact, I'm making it even a little wider angle. There's a reason, I don't want it to encroach up here. We're conjuring something, we're conjuring up an animal that we don't know how big it's going to be.

Now, I take all those lines, we dropped them down, and now we're going to put them into perspective.

Since that front corner is there, it goes back to the left vanishing point at an angle like this. Where does it end? The ground part of it will end right there. That is the part that touches the ground.

Remember, it's important to be precise with these lines. Mistakes can compound, but I correct them as I go.

The next ones are just connecting the dots. If we've done our job right, the points will align correctly when we point them to their respective vanishing points.

Transferring the Heights and Elevations

Now it's time to transfer the heights. We need to measure how high anything is, say, the height of different parts of the house.

There's only one corner that touches the picture plane, which means when we bring it down here, I'm going to darken this line because that's going to be the only non-foreshortened line. The heavy line is the line that we're going to use to take all measurements.

I line up the T-square and pull it tight so that it goes right there. I place the base of this house so it just rests perfectly on that line and burnish it down.

Now, we can transfer the heights from our elevations onto the drawing.

Using Reference Points

We know that the highest point is the peak of the roof. We also have points we're sure of, such as this point right here, and we know where it is not only top to bottom but left to right. It is perfectly there. We should make that a prime reference point.

We need the height of the peak of the small roof and the base of the small roof. The height of these elements is important.

Once we have that height established, we run it back to the left vanishing point. This gives us, with a few connect-the-dots moments, the correct placement on the drawing.

We repeat this process for other elements, using the right vanishing point where appropriate. It's an elaborate but fun process of connecting the dots.

Once we have a basic measurement, we'll find all the other ones.
