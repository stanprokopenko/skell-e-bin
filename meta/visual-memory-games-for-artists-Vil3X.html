Visual Memory Games for Artists

Copy Code

Improving your visual memory is crucial for artists. It helps you draw from imagination, increases your drawing speed, enhances creativity, and aids in drawing from observation. By having a large library of images in your head, you can identify important information and recognize patterns more effectively. Here are several exercises, challenges, and games to help you improve your visual memory.

Visual Memory Exercises

Draw from Memory

  • Look at a reference for a minute or two.
  • Hide the reference and draw as much as you can from memory.
  • Check your accuracy after drawing.
  • Optionally, repeat the exercise to see if you can improve.

Draw from a Different Angle

  • Look at a reference and draw it from a different angle.
  • No need to hide the reference for this exercise.

Combine Memory and Different Angle

  • Look at a reference for a few minutes.
  • Draw from memory and from a different angle.
  • Optionally, start with a study drawing to learn the construction of the forms better.

Advanced Exercise: Change Proportions

  • For experienced artists.
  • Draw from figure reference.
  • Change the proportions while drawing from memory and a different angle.

Bonus Exercise: Draw from a Moving Subject

  • Observe a moving subject, like animals at the zoo.
  • Take a mental snapshot of a pose.
  • Draw the pose from memory while the subject continues to move.

Drawing Basics Course Overview

Level One: Beginners

  • Draw from provided 3D models.
  • Choose from the first three games: draw from memory, draw from a different angle, or combine them.
  • Focus on practicing perspective.
  • Use interactive 3D models for instant feedback.

Level Two: Intermediate Artists

  • Draw from photos of figure poses.
  • Choose from four games, including changing proportions.
  • Simplify poses to a mannequin using gestural torso boxes, cylinders for limbs, and a Loomis head or simple shapes for the head.

Continuous Practice and Community Engagement

  • Keep playing memory games after the course to continue improving your visual memory.
  • Focus on different aspects like gesture, shapes, colors, and shadows.
  • Submit your work to the community for feedback.

* * *

Remember, improving your visual memory takes practice, but these exercises can help you get there faster. Have fun and practice regularly!
