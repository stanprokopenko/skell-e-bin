Why Make Value Scales

Creating value scales teaches you how to control your mediums and develops your patience. If you can't produce exact values in a simple scale, you can't expect to handle complex portrait drawings. It's beneficial to make a value scale for every medium you want to master, whether it's pencils, markers, ink, or charcoal.

Level 1: Basic Value Scale

Start by drawing a 1x1 inch rectangle divided into 11 squares. Feel free to use a ruler, and draw the lines very lightly to avoid dark lines between squares, which can hinder value comparison.

  1. Establish Your Darkest Value: Find the darkest value your pencils can produce. This is your value zero.
  2. Define Your Lightest Value: The white of the paper serves as your lightest value.
  3. Fill in the Middle Values: Gradually fill in the squares between the darkest and lightest values. You can work sequentially or start in the middle and adjust as needed.

Using a range of pencils like 2H, HB, 2B, 4B, and 6B makes this task easier, as each pencil has a comfortable range. If you're using fewer pencils, you'll need better control to achieve the full range of values.

Tips:

  • Err on the Side of Being Too Light: It's easier to darken values than to lighten them.
  • Use a Kneaded Eraser: If you need to lighten an area, blotting with a kneaded eraser can help.
  • Aim for Clean Shading: Avoid holes and inconsistent patches. Revisit and adjust squares to ensure a consistent progression without blending tools.
  • Squint at Your Work: This helps you blur the values together and see where jumps are too extreme or values are too similar.

If you want an easier start, create a scale with five squares. Begin with the extremes, then fill in the middle and the values between.

Level 2: Blind Value Scales

Challenge yourself by making a blind value scale. For this exercise:

  1. Prepare Separate Squares: Cut out individual 1 or 2-inch squares. Number each one on the back.
  2. Shade Each Square Independently: Do each value without seeing the others. You can do them in a specific order or randomly for added difficulty.
  3. Line Them Up: Once finished, arrange them in order to see how smooth the gradation is.

Adjust the difficulty by changing the number of squares. Doing five squares blind is much easier than doing eleven.

Optional: Marker Value Scales

If you're working with markers, try creating a value scale to practice even application:

  • Use Multiple Markers: Combine markers of different shades (e.g., cool gray 1, 3, 5, 7, and black) to create a range of values.
  • Practice Consistent Strokes: Overlapping inconsistently can create striations. Apply strokes evenly to achieve smooth values.
  • Layer for In-Between Values: Apply the same marker multiple times or combine markers to create intermediate values.
  • Be Patient with Drying: Wait a few minutes for the markers to dry to see the true value, as wet pigment appears darker.

Final Advice

Approach these exercises with patience and take pride in your work. Practicing clean and accurate values pays dividends in your drawing skills. As you progress, remember that how you light your work affects the values in your photos, so learn to take evenly lit, high-quality images of your drawings.
