Copy Code

Greg shared an amazing quote on Twitter today: "The most unstoppable person on the planet right now is someone who knows how to create videos that the algorithm loves and someone who knows the new LLM tools like Claude v0 and Cursor." This resonates with me because I've become a creator composer over the last four years.

The Creator Composer Concept

I've been creating content mainly on TikTok and Instagram, where I almost have 1 million followers. I've gotten good at explaining my ideas across platforms. But composing isn't just about content creation, it's about using language to instruct AI tools. The most important language to learn is English to create apps.

Leveraging AI Tools

I've been using a stack of tools: v0, Cursor, Replit, and Firebase.

v0

I use v0 to design sites. By typing in English, it designs the site for me, just like that.

Cursor

Cursor allows me to generate and edit many code files in one prompt using its Composer feature. It's the best software feature I've ever used.

Replit

After generating the code, I use Replit to deploy it. Deploying is harder than it looks, but Replit makes it easier. It's like putting your app on the web.

Firebase

Every app needs a database. I use Firebase to store data like text posts and photos, just like how Twitter stores tweets and images.

Building and Deploying Apps Quickly

These tools make it easier than ever to go from an idea to a deployed app. Four months ago, I didn't know how to do this. But now, with these tools, it's possible to build full apps quickly. We've even created a free template that over 6,000 people have used to build an app.

The Importance of Storytelling

Many people can write software, but not all can get users. That's because storytelling is crucial. Software is becoming so easy and fun to make that it's becoming an extension of your content.

Learning in Public

I'm trying a new strategy called learn in public. It's about picking something new to learn, sharing your progress, and creating content about it. If you're exploring new possibilities, you'll be more valuable because others haven't caught up yet.

Our First App: VoicePower

We're building an app called VoicePower. It uses voice to extract ideas and turn them into good content. It prompts you with ideas, helps you get your thoughts out, and structures them into something polished and consistent.
