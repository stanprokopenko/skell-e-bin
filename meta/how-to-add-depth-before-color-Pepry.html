How to Add Depth Before Color

Organizing Layers

After finishing the sketch, I organize my line work by grouping all line layers (Ctrl/Cmd + G) and flattening them into one layer (Ctrl/Cmd + E). This gives me a clean line work layer to work with.

Laying in Flats

Underneath the line work, I create a new group called "flats". I start blocking in flat colors, no gradients, using the lasso tool. If you have a plugin that fills shapes instantly, that's great, but you can also use the lasso and paint bucket tools. I block out the entire base shape first, and this will serve as the foundation.

Using Clipping Masks

I place all subsequent layers within the base shape using clipping masks. By right-clicking a layer and selecting "Create Clipping Mask" or holding Alt while clicking between layers, I ensure all my painting stays within the base shape. This method allows me to stack layers while keeping everything contained.

Defining Local Values

I add local values for different materials. For example, the eyes and teeth are lighter than the skin. I quickly block in these areas, knowing we can refine them later. This step brings more clarity to the piece.

Adding Shadows with Multiply Layers

To add depth, I focus on the shadows. I create a new layer with the Multiply blending mode, which darkens areas while retaining the underlying colors. Using the lasso tool, I define shadow shapes based on my top-down light source. I darken areas like under the nose, the inner mouth, and the eyebrows.

Simulating Light Fall-Off

I use a soft round brush to simulate light fall-off, with an overall shadow pass. This technique directs the viewer's focus to the focal point, in this case, the face. By darkening areas away from the light, I enhance the three dimensional effect.

Refining Values and Merging Layers

Satisfied with the shadows, I merge the layers for more control. I refine and adjust the values, smoothing transitions and enhancing forms. Turning the line work on and off helps me see how the values stand on their own.

Adding Highlights with Screen Layers

To introduce highlights, I use a layer set to the Screen blending mode. This lightens areas while keeping the underlying context. I carefully add lighter values where the light hits directly, such as the brow, nose, and cheeks, enhancing the dimensionality.

Iterating and Adjusting

I continuously squint down to check the values, ensuring the lights and shadows work together. I adjust features as needed, sometimes repositioning elements like the eyes. Merging layers allows me to manipulate the painting freely, pushing and pulling shapes to improve the piece.

Preparing for Color

With the values in place, the piece is ready for color. Focusing on getting the values right in black and white makes adding color much more straightforward. Next, I'll start applying color on top of these established values to bring the piece to life.
