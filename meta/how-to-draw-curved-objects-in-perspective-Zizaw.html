How to Draw Curved Objects in Perspective

I've been studying Blondie the Good's arrow drawing, and I've found many insights worth sharing. It dives deep into the structure of an arrow in space, and I think it's worth spending a session imagining and solving the technical challenges it presents.

Understanding the Structure

Let's start with the drawing itself. I printed it out and added some annotations, hoping to respect the original work. We know the space where the arrow fits because Blondie fixed it within a box. Whether the arrow tapers or not, we can't tell just yet. I propose that it does not taper.

It might look like it's getting thinner in some areas, but I believe the lines of the arrow can be tracked on either side, touching a constant-width box. We've got to address why it appears to be thinning in the middle.

To illustrate this, let's exaggerate the effect. Now it seems even thinner, right? But in reality, it's only thinner on an axis that isn't relevant to our analysis. If we stick with X lines in this drawing, they will all be the same. That's one reason we draw width lines: to keep track of the width of the object, not just our perception of it, until we've learned to see these things intuitively.

The Importance of Width Lines

The X lines may change width if they change on the object, or if we choose to put it in extreme perspective, that rapidly approaching wide-angle lens effect. In this drawing by Blondie the Good, the curve can confound our ability to be sure of the width until we understand what is going on.

By the way, someone commented on the dullness of my pencil and felt the urge to sharpen it. I'm choosing a non-sharp pencil on purpose. It makes a thicker line, which you can see more easily, especially if you're watching this on your smartphone. No judgment, it's a convenient way to learn.

Analyzing the Arrow in Orthographic View

The curve in the arrow is only an up and down curve, it does not twist like Lucy's arrow. Not twisting should make this simpler. Let's go as simply as we can to the ortho, which we can deduce because the arrow is in a box.

We're creating the ground plane with just a hint of the box, and we're going to see what the arrow would look like if we were looking straight down from above. How do we know where a point on the arrow is on the ground? Since that is a right angle, we can draw a vertical line straight down, touch that point, get the base of the box, and then run the same line system out on the Z axis.

Let's do this for all the relevant points. By connecting these points, we can draw the arrowhead on the ground as a flat arrow. We could have done this on the top of the box or just gone to the ortho view if we were working from one, and see that it doesn't taper.

Dealing with Curves and Pitch

Even though the arrow curves, it only curves on one axis, down to up. It moves straight forward once it has lifted off the ground plane. One way to clarify this is to place a light source directly above it, emphasizing that there's no messing about with width.

We need to name that curve. The technical name for movement that only goes down or up as it moves forward and back is pitch. Think of it like the movement seen in a side view, similar to a musical staff showing the pitch of notes. That's what's happening with this curve, it goes down, then gently up, but not as far up as it went down. Then, very importantly, it goes straight again on a pure Z-line.

This defines the kind of movement that the arrow is making.

Simplifying Perspective by Thickening the Arrow

Perspective can be confusing, so when learning it, it's good to make things so clear they can't be misread. Let's thicken the arrow by extruding it upward. I'm extruding it up rather than down. Either way creates surprises.

As we extrude the arrow, we need to be careful with the lines. For example, when I carry a line all the way through to the front, I shouldn't, because it will be behind another part of the arrow. We have to be cautious not to darken lines until we're sure where they stop.

Adding Taper to the Arrow

Now, let's take on the challenging task of making the arrow curve and also taper as it moves forward. I'll show you how.

First, let's review, the pitch moves only on Y and Z, down on Y as it moves forward on Z, then up on Y as it moves forward on Z. If we taper the arrow, we'll add movement on the X-axis. We're creating a complexity that we shouldn't begin with, in my opinion.

We should start by finding the flat ortho view on the top plane, the same way we found it on the bottom, and be sure of the points where it touches on the sides. We can decide how much to narrow the arrow in the top view. Let's say we bring it in a certain amount on one side, and match that amount on the other side to keep it symmetrical. We're audaciously choosing a taper.

Solving the Technical Challenge

To solve this without ambiguity, we need to make note of how much the straight arrow tapers on the X-axis in the plan view. We establish the widths without the complication of a curve.

Next, we find the height from the curve and use the simplicity of right angles to run the width of that height across, and the height of that width down. By combining these methods, we can find precise points along the arrow's curve that account for both the taper and the curve.

By plotting these points, we can draw the arrow with an accurate compound curve. This method ensures that the curve is not only visually smooth but also technically correct.

The Difficulty of Compound Curves

Compound curves are difficult. I'm not putting this on you to intimidate you, but if you really care, what I've shown you cracks the code.

Getting these curves technically correct may not be the best idea for an artist. Let me explain why. When we aim for pure accuracy, we might lose the expressive quality that's important in art.

For example, if we taper the arrow with straight lines from the top, it might not capture the essence of the curve. If the sides tapered concavely, it would accentuate the barrel roll on our side but contradict it on the other, which could look awkward or at least damage the illusion.

A decisive artist, not bound by technical correctness, could choose to cheat, the way filmmakers cheat when they shoot scenes in separate environments and piece them together to seem like one. There's no moral component to that kind of cheating. There's a higher calling, which is to create a worthy illusion.
