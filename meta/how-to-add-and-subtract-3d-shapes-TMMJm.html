How to Add and Subtract 3D Shapes

I'm gonna do the memory game, drawing from memory. I'm drawing a whistle. Before that, I'll show how to add and subtract 3D primitives. Many objects are built from basic volumes combined. Organic things are harder to fit into primitives but still can be. For example, a camera is a box with a cylinder for the lens. A pen is a cylinder with various boxy and cylindrical parts. Sometimes you need to subtract one shape from another. I'll show how to do that.

Adding 3D Primitives

Let's start with a box and add an elongated cylinder on top. Here's how to do it:

  1. Draw an ellipse lightly to get a decent shape.
  2. Carry it over to the other side and draw a similar ellipse.
  3. Darken the lines to indicate the outer form.
  4. Erase construction lines to clarify the final drawing.

Now it looks like a box with a rounded part on top. This shows how to build complex things from simple shapes.

Subtracting 3D Primitives

Now let's do subtraction. Start light and rotate the sketch pad. Here's the process:

  1. Draw another box and subtract a shape from the bottom.
  2. Draw an oval at a different angle and carry it over to the other side.
  3. Darken lines to subtract the shape, creating a valley.

Think of the entire volume when subtracting.

Drawing a Whistle from Memory

Now, let's try the whistle. I'll stare at it, memorize, and then draw from memory. In class, we did figure drawings from memory, looking for a minute and drawing for 3-5 minutes. Here's the process:

  1. Identify basic volumes: the body is a cylinder, and the mouthpiece is a tapering box.
  2. Note the subtraction at the top, a little forward.
  3. Hide the whistle and start drawing.
  4. Start with the top angle of the box, then taper it.
  5. Add the cylinder, drawing a square and an ellipse.
  6. Darken lines to define the shape.
  7. Add the little notch at the top, drawing a smaller oval.
  8. Use construction lines to get the cutout right.
  9. Add thickness and details.
  10. Compare with the reference image.

The angle was off, but the whistle looks similar. This exercise improves visual memory and helps in constructing shapes.
