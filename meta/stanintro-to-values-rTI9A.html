What is Value in Art?

In art, value refers to how light or dark something is. It's not about worth or cost. If you take a photo and make it grayscale, you'll notice differences in values—the skin might be lighter than the lips, but sometimes different colors can have similar values.

Lines vs. Values

While we've learned a lot about lines, remember that lines aren't real. We don't actually see lines; we see light affecting forms. Lines are an abstraction we use to represent the edges of shapes. In reality, what we see is an arrangement of colors filling shapes.

The Components of Color

Color consists of hue, chroma, and value.

Hue, Chroma, and Value

  • Hue is the color family—yellow, red, blue, etc.
  • Chroma describes how pure or gray a color is.
  • Value is the lightness or darkness of a color, from white to black.

Value is Key

While hue and chroma are important, value is the most crucial component for creating the illusion of form. Artists often focus on value separately because it plays a key role in depicting light and form.

Values Create Form and Depth

Think of a drawing like a building:

  • Lines are like the steel beams.
  • Values are like the walls filling in the spaces, making it solid.
  • Textures and colors are the decorations and details.

Values help create the illusion of form, depth, and volume. As a beginner, it's wise to practice drawing without color, focusing on values to understand light and form.

The Value Scale

To organize values practically, imagine a scale from 0 (darkest black) to 10 (brightest white).

Using the Munsell System

We'll use the Munsell system, which measures light rather than pigment. A value of 10 means an area is getting the most light, while 0 means it's in complete shadow.

Simplifying the Value Scale

In reality, there's an infinite gradation between black and white, but it's impractical to work with infinity. So, we break it down into manageable steps, usually about 10 values. This simplification makes it easier to communicate and work with values.

Determining the Values We See

There are four factors that determine the values we see:

  1. Light on Form
  2. Local Value
  3. Surface Material
  4. Atmosphere

As a beginner, focus on the first two.

Light on Form

Light affects how we perceive the value on an object's surface. Areas facing the light source appear lighter, while planes turning away appear darker.

Local Value

Local value is the inherent lightness or darkness of an object, regardless of lighting. For example, hair is generally darker than skin. An object's local value isn't always consistent; it can vary across its surface.

Focusing on Values as a Beginner

Both light on form and local value work together to create the values we see. By concentrating on these, you'll train your eye to notice subtle shifts in value and translate them into your drawings. Practicing with values before introducing color will have the biggest impact on your art.
